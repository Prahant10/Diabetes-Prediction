Diabetes Prediction Model
A machine learning project that predicts whether a patient is diabetic based on health metrics, using a Support Vector Machine (SVM) classifier.
Dataset
Uses the Pima Indians Diabetes Dataset, which contains 768 records with 8 medical features:

Pregnancies
Glucose level
Blood pressure
Skin thickness
Insulin level
BMI
Diabetes pedigree function
Age

Model

Algorithm: Support Vector Machine (SVM) with a linear kernel
Preprocessing: StandardScaler for feature normalization
Train/test split: 80% train, 20% test
Accuracy: ~77–80%
